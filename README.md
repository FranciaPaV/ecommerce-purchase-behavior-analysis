# E-commerce Purchase Behavior Analysis

## Objective
Explore customer purchase behavior to identify shopping patterns and product trends in an e-commerce platform.

## Context
Understanding how customers shop helps improve product strategy, inventory planning and marketing decisions.

## Data
Transactional data from an e-commerce platform, including:
- Orders
- Products
- Reorders
- Purchase timing and frequency

## Analysis
The analysis focused on:
- Order frequency and timing
- Most purchased and reordered products
- Shopping behavior patterns across customers

## Tools
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

## Key Insights
- Certain products show strong reorder behavior
- Purchase timing reveals clear customer habits
- Behavioral insights can inform promotions and inventory decisions
